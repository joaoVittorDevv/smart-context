# Frontend & UI Iteration Skill

1. **Análise de Contexto:** Analise os componentes existentes para manter a consistência visual.
2. **Ciclo de Desenvolvimento:**
   - Após codificar, levante o servidor de dev no Terminal.
   - Utilize o Browser integrado do Antigravity para testar.
3. **Feedback Visual:** Tire um screenshot (como um Artifact) para mostrar ao usuário como ficou.
4. **Iteração:** Ajuste iterativamente baseado no feedback visual fornecido pelo usuário.
